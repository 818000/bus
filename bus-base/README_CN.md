# 📦 Bus Base: Bus 框架基础模块

<p align="center">
<strong>提供通用工具和配置的基础模块</strong>
</p>

-----

## 📖 项目介绍

**Bus Base** 是 Bus 框架的基础模块，为所有其他 Bus 模块提供通用工具、共享配置和基础功能。它是整个生态系统的构建基石。

-----

## ✨ 核心特性

- **通用工具**: 跨模块共享的工具类和辅助方法
- **基础配置**: 所有 Bus 模块的默认配置
- **常量定义**: 集中管理的常量和枚举
- **异常层次结构**: 用于错误处理的基础异常类
- **共享接口**: 跨模块使用的通用接口

-----

## 🚀 使用方式

### Maven 依赖

```xml
<dependency>
    <groupId>org.miaixz</groupId>
    <artifactId>bus-base</artifactId>
    <version>x.x.x</version>
</dependency>
```

### 自动包含

**注意**: 此模块通常作为传递依赖自动包含在使用其他 Bus 模块时。您通常不需要显式添加它。

-----

## 🔧 组件

### 通用工具

- 字符串工具
- 集合工具
- 日期/时间工具
- IO 工具
- 验证工具

### 基础配置

- 默认字符集设置
- 区域设置配置
- 时区设置
- 日志配置

### 共享接口

- 可序列化标记
- 生命周期接口
- 配置接口
- 提供者接口

-----

## 💡 使用场景

- 作为自定义 Bus 模块的基础
- 跨模块访问共享工具
- 标准化配置管理
- 提供通用异常处理
